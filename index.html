HTML5 Web Storage

This allows web applications to store data locally in the user's browser, which is more secure and can store large amounts of data.

Document Object Model (DOM)

This is a programming interface for web documents that allows programs to change the document's structure, style, and content.

JavaScript

This high-level, interpreted programming language is used to create interactive web pages and online apps with dynamic functionality.

Scope in JavaScript

This describes how accessible functions and variables are in the running environment. JavaScript supports two scopes, including local scope, where values and functions declared within the same function can only be accessed within that function.
